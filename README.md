## 🗺️ Roadmap de Aprendizaje

| Fase | Meses | Enfoque |
|------|-------|---------|

| 2 | 4-6 | Bases de datos relacionales |
| 3 | 7-9 | Estructuras discretas y algoritmos |
| 4 | 10-14 | Desarrollo Full-Stack |
| 5 | 15-18 | Estadística y análisis de datos |
| 6 | 19-21 | Bases de datos avanzadas y NoSQL |
| 7 | 22-24 | Arquitectura y especialización |
