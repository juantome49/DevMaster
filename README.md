# DevMaster Academy 🚀

Plataforma web educativa interactiva y profesional para aprender **Programación**, **Bases de Datos**, **Matemáticas Computacionales** y **Análisis de Sistemas**.

## 📚 Contenido Educativo

### 🗄️ Bases de Datos
- **Básico**: SQL CRUD, Modelado ER, JOINs, Normalización
- **Intermedio**: Window Functions, Optimización, Transacciones ACID, Stored Procedures
- **Avanzado**: NoSQL (MongoDB, Redis, Cassandra), Sharding, Data Warehousing, Sistemas Distribuidos

### 📐 Matemáticas Computacionales
- **Básico**: Lógica proposicional, Sistemas numéricos, Teoría de conjuntos, Funciones
- **Intermedio**: Matemáticas discretas, Probabilidad y estadística, Álgebra lineal, Teoría de grafos
- **Avanzado**: Cálculo diferencial/integral, Análisis de algoritmos (Big O), Criptografía, ML matemático

### 💻 Programación
- **Básico**: Python, HTML5/CSS3/JS, Estructuras de datos básicas
- **Intermedio**: React, Node.js, Java Spring Boot, Testing TDD
- **Avanzado**: Arquitectura de software, Machine Learning, Ciberseguridad, DevOps

### 📊 Análisis de Sistemas
- **Básico**: Levantamiento de requisitos, Modelado de procesos, BPMN
- **Intermedio**: UML avanzado, Análisis de negocio, KPIs
- **Avanzado**: Arquitectura empresarial (TOGAF), Gestión del cambio, Transformación digital

## 🎯 Características

- ✅ **Niveles progresivos**: Básico → Intermedio → Avanzado
- ✅ **Laboratorio interactivo**: Playgrounds para SQL, Algoritmos y Matemáticas
- ✅ **Visualizaciones**: Diagramas ER, fórmulas matemáticas, trazas de ejecución
- ✅ **Roadmap integrado**: Ruta de aprendizaje de 24 meses
- ✅ **Sistema de progreso**: Tracking de avance por área
- ✅ **Diseño responsive**: Adaptable a móviles y tablets
- ✅ **Tema dark moderno**: Interfaz profesional con gradientes cyberpunk

## 🚀 Uso

1. Descarga o clona el repositorio
2. Abre el archivo `index.html` en tu navegador
3. ¡Comienza a aprender!

No requiere instalación ni dependencias externas. Funciona completamente en el cliente.

## 🛠️ Tecnologías

- **Frontend**: HTML5 semántico, CSS3 (Grid, Flexbox, Animaciones), JavaScript vanilla
- **Diseño**: Tema dark con paleta cyberpunk (cyan, magenta, púrpura, verde menta, amarillo)
- **Tipografías**: Inter (UI), JetBrains Mono (código)
- **Interactividad**: Tabs dinámicos, playground de código simulado, animaciones CSS/JS

## 🗺️ Roadmap de Aprendizaje

| Fase | Meses | Enfoque |
|------|-------|---------|
| 1 | 1-3 | Fundamentos matemáticos y lógica |
| 2 | 4-6 | Bases de datos relacionales |
| 3 | 7-9 | Estructuras discretas y algoritmos |
| 4 | 10-14 | Desarrollo Full-Stack |
| 5 | 15-18 | Estadística y análisis de datos |
| 6 | 19-21 | Bases de datos avanzadas y NoSQL |
| 7 | 22-24 | Arquitectura y especialización |
