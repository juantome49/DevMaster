Ejemplo de Modelo Relacional: Sistema E-commerce

USUARIOS

  • PK id_usuario INT
  • nombre VARCHAR(100)
  • email VARCHAR(100)
  • fecha_registro DATE
→

ORDENES

  • PK id_orden INT
  • FK id_usuario INT
  • fecha_orden TIMESTAMP
  • total DECIMAL(10,2)
→

DETALLE_ORDEN

  • PK id_detalle INT
  • FK id_orden INT
  • FK id_producto INT
  • cantidad INT
  • precio_unit DECIMAL
→

PRODUCTOS

  • PK id_producto INT
  • nombre VARCHAR(200)
  • precio DECIMAL(10,2)
  • stock INT
